# Next.js with Material UI, Next-PWA and Zustand

Template for Next.js with Material UI, Next-PWA and Zustand.

## Requirements

- Node.js
- npm or yarn

## Running locally

```
npm install or yarn install
```

### Running development environment

```
// with npm
npm run dev

// with yarn
yarn dev
```

### Running production environment

```
// with npm
npm run prod

or

// with yarn
yarn prod
```
